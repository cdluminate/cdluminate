Quick Access to Debian Services Toggle Usage

Usage 1: Input your query (e.g. apt) and click a button on the right side.

Usage 2: Input a IDENTIFIER:QUERY pattern and press Enter, for instance t:apt. Identifier t: stands for tracker, while bd: for buildd, bug: for BTS, ci: for CI, uci: for Ubuntu's CI. When the identifier is omitted, you will be directed to the package tracker.

Note, any identifier will override the actuall button you clicked.
